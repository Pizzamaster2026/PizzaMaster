# 🍕 PizzaMaster

Willkommen bei PizzaMaster!

PizzaMaster ist ein moderner Pizzateig-Rechner für Hobby- und Profi-Pizzabäcker.

## Funktionen

- 🍕 Pizzateig berechnen
- 💧 Hydration berechnen
- 🧂 Salz berechnen
- 🍺 Frisch- und Trockenhefe
- 📱 Optimiert für Smartphones
- 🌙 Dark Mode
- 🚀 Installierbar als App (PWA)
